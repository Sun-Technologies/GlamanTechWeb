Core SAP applications to be certified on Microsoft Azure Cloud

Microsoft Azure is now certified for SAP business applications. SAP apps are available in Microsoft’s cloud. This will help consumer to simplify infrastructure management, improve time to market, and lower costs.

Microsoft will support customers if a problem crops up at the infrastructure level, while SAP would take over if the issue involves an application error, said Kevin Ichhpurani, senior vice president, head of business development and strategic ecosystem at SAP.

Oracle Database In-Memory option something to remember

Oracle Database In-Memory was released today at Oracle headquarters in Redwood Shores, California. It is an add-on, and will be available on all hardware platforms supported by Oracle Database 12c.

The Oracle Database In-Memory option has a new fault tolerance system also designed around Real Application Clusters.

Oracle Database In-Memory option's compression capabilities allows the Oracle Database to keep very large tables in compressed memory, said Christo Kutrovsky, senior consultant with Python group.
